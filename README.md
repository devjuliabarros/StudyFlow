<h2>| Sobre o Projeto </h3>
<img src="public/assets/imgs/readme/logo.png" alt=""> <br>
O <b>Study Flow</b> é um sistema web de planejamento de estudos que auxilia estudantes na organização de tarefas, controle de tempo e acompanhamento de produtividade. <br>

<h2>| Funcionalidades </h3>
🔐 Cadastro e login de usuários <br>
📊 Dashboard com indicadores de progresso <br>
📚 Cadastro de tarefas de estudo<br>
📅 Visualização de tarefas do dia <br>
✔️ Marcar tarefas como concluídas <br>
🗑️ Exclusão de tarefas <br>

<h2>| Como Usar </h2>
1. Clone o repositório:
<b>https://github.com/devjuliabarros/StudyFlow.git </b> <br>
2. Acesse o diretório do projeto no terminal com o comando <b> cd StudyFlow/</b> <br>
3. Execute o comando<b>  npm start </b> <br>
4. Acesso o endereço do servidor <b>http://localhost:3333/</b> no seu navegador <br> <br>
